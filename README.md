## Core Services
AWS 에서는 무수히 많은 서비스가 존재하며 끊임없이 새로운 서비스가 만들어진다.  
현재(2021년) AWS는 175 이상의 서비스를 제공한다. 이 모든 것을 깊이 있게 학습할 필요는 없지만, 아래 AWS의 핵심적인 서비스들은 심층적으로 탐구해야 한다.  

- Auto Scaling
- Availability Zones
- EC2
- ELB
- Regions
- Storage
- Networking
- VPC
